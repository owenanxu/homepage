# Owen Xu Homepage

Personal homepage for Owen Xu, presenting a profile as an AI scientist working at the intersection of machine learning and biology.

## Overview

This is a lightweight static website built with plain HTML, CSS, and JavaScript. It includes:

- A responsive homepage hero
- About, work, and contact sections
- Smooth in-page navigation
- Automatic footer year update

## Files

- `index.html` - page content and structure
- `style.css` - visual design and responsive layout
- `script.js` - small browser enhancements

## Preview

Open `index.html` directly in a browser, or serve the folder with a simple static server:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
